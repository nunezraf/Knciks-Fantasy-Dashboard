The Project

The "NBA Fanyasy Basketball Project" was done with the purpose of helping aspiring fantasy basketball players navigate the wild world of fantasy. All the rosources provided are aimed to speed the knowledge process and help the players to be as good as the veteran. However, this project also has the nessceasry tools to help veteran fantasy players to become even better. this dashboard provides visual analysis, top news, players raking, top fantasy basketball websites and the ability of having a main dashboard capable of being used as the got to site for fantasy.
